# 区块链书籍资源汇总 

不管虚拟货币牛市抑或熊市，区块链技术越来越火爆，作为技术人员，如何掌握新的技术趋势？这里分门别类整理了区块链相关的书籍，有助于你更深入、系统、全面的了解区块链技术。

网络上有各种各样的入门教程，但是再多的博客、教程都比不上一本区块链的书籍来的高效，所以还是推荐大家通过区块链书籍来进入区块链世界。
